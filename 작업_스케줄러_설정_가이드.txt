# Windows 작업 스케줄러 설정 가이드

## 📋 작업 스케줄러 설정 방법

### 1단계: 작업 스케줄러 열기
- `Win + R` → `taskschd.msc` 입력
- 또는 시작 메뉴에서 "작업 스케줄러" 검색

### 2단계: 기본 작업 만들기
1. **기본 작업 만들기** 클릭
2. **이름**: `EIMS 자동화 시스템`
3. **설명**: `컴퓨터 부팅 시 EIMS 스케줄러 자동 시작`

### 3단계: 트리거 설정
1. **컴퓨터 시작할 때** 선택
2. **지연**: `2분` (시스템 안정화 대기)

### 4단계: 작업 설정
1. **프로그램 시작** 선택
2. **프로그램/스크립트**: `C:\Users\USER\Desktop\삼양자동화\boot_start_eims.bat`
3. **시작 위치**: `C:\Users\USER\Desktop\삼양자동화`

### 5단계: 조건 설정
- ✅ **컴퓨터가 절전 모드에 있을 때 작업 중지**: 체크 해제
- ✅ **네트워크가 사용 가능할 때만 시작**: 체크 해제

## 🎯 결과
- 컴퓨터 부팅 → 2분 후 → EIMS 스케줄러 시작
- 매일 오전 9시 30분, 오후 2시에 자동 실행
