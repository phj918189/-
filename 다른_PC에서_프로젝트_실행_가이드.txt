# 삼양자동화 프로젝트 - 다른 PC에서 실행하기 가이드

## 📋 개요
현재 삼양자동화 프로젝트는 이 PC에서만 실행되고 있습니다. 
다른 PC(집, 랩탑 등)에서도 동일하게 실행하려면 다음 단계를 따라주세요.

## 🔍 현재 프로젝트 상태 분석
- **프로젝트 위치**: C:\Users\USER\Desktop\삼양자동화
- **주요 기술**: Python, Playwright, SQLite, 웹 스크래핑
- **자동화 기능**: EIMS 시스템 로그인 → 데이터 다운로드 → 작업 배정 → 이메일 알림
- **실행 방식**: 배치 파일(.bat) 또는 Python 스크립트 직접 실행

## 🚀 다른 PC에서 실행하는 방법

### 방법 1: 전체 프로젝트 복사 (권장)

#### 1단계: 프로젝트 폴더 복사
```
1. 현재 PC의 프로젝트 폴더 전체를 복사
   - 원본: C:\Users\USER\Desktop\삼양자동화
   - 대상: 다른 PC의 원하는 위치 (예: C:\삼양자동화)

2. 복사할 폴더들:
   ✅ scripts/          (핵심 Python 스크립트)
   ✅ sql/              (데이터베이스 스키마, 설정 파일)
   ✅ storage/          (데이터베이스, 로그, 다운로드 파일)
   ✅ config.yaml       (설정 파일)
   ✅ *.bat             (실행용 배치 파일)
   ✅ *.py              (루트 레벨 Python 파일)
   ✅ image/            (이미지 파일들)
   ✅ *.txt, *.md       (문서 파일들)
```

#### 2단계: Python 환경 설정
```
1. Python 설치 확인
   - Python 3.8 이상 필요
   - 명령 프롬프트에서 확인: python --version

2. 필요한 패키지 설치
   - 프로젝트 폴더에서 명령 프롬프트 실행
   - 다음 명령어 실행:
     pip install -r sql/requirements.txt
   
   또는 개별 설치:
     pip install playwright pandas python-dotenv SQLAlchemy openpyxl requests
     playwright install chromium
```

#### 3단계: 환경 변수 설정
```
1. .env 파일 생성 (프로젝트 루트에)
   - EIMS_ID=당신의_로그인_ID
   - EIMS_PW=당신의_비밀번호
   
2. 이메일 설정 (선택사항)
   - EMAIL_PASSWORD=이메일_앱_비밀번호
   - EMAIL_SMTP_HOST=smtp.gmail.com
   - EMAIL_SMTP_PORT=587
```

#### 4단계: 실행 테스트
```
1. 수동 실행 테스트
   - 명령 프롬프트에서: python sql/job.py
   - 또는 배치 파일 실행: run_eims_auto.bat

2. 스케줄러 설정 (자동 실행)
   - start_scheduler.bat 실행
   - 또는 Windows 작업 스케줄러에 등록
```

### 방법 2: 클라우드 동기화 (고급)

#### 1단계: 클라우드 폴더 설정
```
1. OneDrive, Google Drive, Dropbox 등 사용
2. 프로젝트 폴더를 클라우드 폴더에 배치
3. 모든 PC에서 동일한 클라우드 폴더 사용
```

#### 2단계: 경로 설정 수정
```
1. config.yaml 파일 수정
   - 절대 경로를 상대 경로로 변경
   - 또는 환경변수 사용

2. 배치 파일 경로 수정
   - 각 PC의 실제 경로에 맞게 수정
```

## ⚠️ 주의사항 및 문제 해결

### 1. 경로 문제
```
문제: 파일 경로가 맞지 않음
해결: 
- 모든 경로를 상대 경로로 변경
- 또는 각 PC에서 config.yaml 수정
```

### 2. Python 패키지 문제
```
문제: 패키지가 설치되지 않음
해결:
- pip install --upgrade pip
- pip install -r sql/requirements.txt --force-reinstall
- playwright install chromium --force
```

### 3. 권한 문제
```
문제: 파일 접근 권한 오류
해결:
- 관리자 권한으로 명령 프롬프트 실행
- 프로젝트 폴더 권한 확인
```

### 4. 네트워크 문제
```
문제: EIMS 시스템 접근 불가
해결:
- 방화벽 설정 확인
- VPN 사용 시 VPN 연결 확인
- 프록시 설정 확인
```

## 🔧 고급 설정

```
방법 1: 파일 락 사용
- 실행 중인 PC에서 lock 파일 생성
- 다른 PC에서 lock 파일 확인 후 실행 중단

방법 2: 데이터베이스 락 사용
- SQLite 데이터베이스 락 활용
- 동시 실행 시 오류 처리
```

### 2. 데이터 동기화
```
방법 1: 공유 네트워크 드라이브
- 모든 PC가 접근 가능한 네트워크 드라이브 사용
- 데이터베이스 파일을 공유 위치에 저장

방법 2: 클라우드 동기화
- OneDrive, Google Drive 등 사용
- 실시간 동기화로 데이터 일치성 유지
```

### 3. 원격 실행
```
방법 1: 원격 데스크톱
- Windows 원격 데스크톱 사용
- 다른 PC에서 원격으로 실행

방법 2: SSH (Linux/Mac)
- SSH를 통한 원격 실행
- Windows에서는 WSL 사용
```

## 📱 모바일/태블릿에서 확인하기

### 해결 방안
1. **웹 대시보드 구축** (권장)

   - 모바일 최적화된 웹 인터페이스
   - 어디서든 브라우저로 접근 가능

2. **클라우드 서비스 활용**
   - Google Sheets 연동
   - Microsoft 365 연동
   - 실시간 공유 가능

## 🎯 권장 실행 방법

### 집에서 실행하기 (가장 간단)
```
1. 프로젝트 폴더 전체를 USB나 클라우드로 복사
2. 집 PC에 Python 설치
3. 필요한 패키지 설치
4. .env 파일 설정
5. 실행 테스트
```

### 랩탑에서 실행하기
```
1. 프로젝트 폴더를 랩탑으로 복사
2. Python 환경 설정
3. 네트워크 연결 확인 (EIMS 접근용)
4. 실행 및 테스트
```

### 여러 PC에서 동시 사용
```
1. 공유 네트워크 드라이브 설정
2. 데이터베이스 파일을 공유 위치에 저장
3. 실행 락 메커니즘 구현
4. 각 PC에서 실행 시도
```
