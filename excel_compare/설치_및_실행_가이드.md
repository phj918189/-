# 엑셀 비교 뷰어 - 설치 및 실행 가이드

## 🎯 빠른 시작 (3단계)

### 1단계: 파일 다운로드 및 압축 해제

1. GitHub에서 ZIP 파일을 다운로드하거나 `git clone`으로 프로젝트를 받습니다
2. 원하는 위치에 압축을 해제합니다 (예: `C:\Projects\excel_compare\`)

### 2단계: Python 설치 확인

**PowerShell**을 관리자 권한으로 실행하고:

```powershell
python --version
```

Python이 설치되어 있지 않다면:
- [python.org](https://www.python.org/downloads/) 접속
- 최신 Python 3.x 버전 다운로드 및 설치
- 설치 시 **"Add Python to PATH"** 옵션 체크 필수!

### 3단계: 패키지 설치 및 실행

```powershell
# 프로젝트 폴더로 이동
cd C:\Projects\excel_compare

# 가상환경 생성 (선택사항이지만 권장)
python -m venv .venv

# 가상환경 활성화
.\.venv\Scripts\Activate.ps1

# 실행 정책 오류 시 (한 번만 실행)
Set-ExecutionPolicy -Scope Process RemoteSigned

# 패키지 설치
pip install --upgrade pip
pip install -r requirements.txt

# 앱 실행
python excel_compare_app.py
```

## 📦 EXE 파일로 실행하기 (Python 설치 불필요)

`dist/excel_compare_app.exe` 파일을 더블클릭하면 됩니다.

**주의사항:**
- 첫 실행 시 Windows Defender가 차단할 수 있습니다
- "추가 정보" → "실행" 클릭
- 또는 Windows 보안 설정에서 예외 추가

## 🔧 상세 설치 가이드

### Windows 10/11에서 Python 설치하기

1. **Python 다운로드**
   - https://www.python.org/downloads/ 접속
   - "Download Python 3.x.x" 클릭

2. **설치 과정**
   - 다운로드한 설치 파일 실행
   - **중요**: "Add Python to PATH" 체크박스 반드시 선택!
   - "Install Now" 클릭
   - 설치 완료 후 PowerShell 재시작

3. **설치 확인**
   ```powershell
   python --version
   pip --version
   ```

### 가상환경 사용하기 (권장)

가상환경을 사용하면 시스템 Python과 충돌 없이 프로젝트별로 패키지를 관리할 수 있습니다.

```powershell
# 가상환경 생성
python -m venv .venv

# 활성화 (PowerShell)
.\.venv\Scripts\Activate.ps1

# 활성화 (명령 프롬프트)
.venv\Scripts\activate.bat

# 비활성화 (언제든지)
deactivate
```

### 패키지 설치 문제 해결

**문제 1: "pip가 인식되지 않습니다"**
```powershell
# Python이 PATH에 추가되지 않은 경우
# Python 재설치 시 "Add Python to PATH" 체크
# 또는 수동으로 PATH 추가 필요
```

**문제 2: "권한 오류"**
```powershell
# 관리자 권한으로 PowerShell 실행
# 또는 사용자 디렉토리에 설치
pip install --user -r requirements.txt
```

**문제 3: "패키지 설치 실패"**
```powershell
# pip 업그레이드
python -m pip install --upgrade pip

# 캐시 클리어 후 재설치
pip cache purge
pip install -r requirements.txt --no-cache-dir
```

## ✅ 설치 확인 체크리스트

- [ ] Python이 설치되어 있고 `python --version` 명령어가 작동함
- [ ] `pip --version` 명령어가 작동함
- [ ] `requirements.txt`의 모든 패키지가 설치됨 (`pip list`로 확인)
- [ ] `python excel_compare_app.py` 명령어로 앱이 실행됨
- [ ] 엑셀 파일을 선택하고 비교 기능이 정상 작동함

## 🆘 자주 발생하는 문제

### 1. "tkinter 모듈을 찾을 수 없습니다"

**원인**: Python 설치 시 tkinter가 포함되지 않았거나, 시스템 Python이 아닌 다른 Python을 사용 중

**해결**:
```powershell
# Python 재설치 (표준 설치 프로그램 사용)
# 또는 tkinter 별도 설치 (Linux의 경우)
```

### 2. "tksheet 모듈을 찾을 수 없습니다"

**원인**: 패키지가 설치되지 않음

**해결**:
```powershell
pip install tksheet
```

### 3. "폰트를 찾을 수 없습니다"

**원인**: Windows에서 맑은 고딕 폰트가 없거나, 다른 OS 사용

**해결**: `excel_compare_config.json`에서 폰트 설정 변경

### 4. 앱이 실행되지만 창이 안 보임

**원인**: 창이 화면 밖에 있거나 최소화됨

**해결**: 
- `excel_compare_config.json` 파일 삭제 후 재실행
- 또는 설정 파일에서 `window.x`, `window.y` 값을 `null`로 변경
