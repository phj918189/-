# 빠른 시작 가이드

## 🚀 3분 안에 시작하기

### 방법 1: EXE 파일 사용 (가장 쉬움)

1. **ZIP 파일 다운로드** 및 압축 해제
2. **`dist/excel_compare_app.exe`** 더블클릭
3. 끝! (Python 설치 불필요)

> ⚠️ 첫 실행 시 Windows Defender가 차단할 수 있습니다. "추가 정보" → "실행" 클릭하세요.

### 방법 2: Python으로 실행

1. **Python 설치** (없는 경우)
   - https://www.python.org/downloads/
   - 설치 시 **"Add Python to PATH"** 체크 필수!

2. **ZIP 파일 다운로드** 및 압축 해제

3. **PowerShell에서 실행**:
   ```powershell
   cd C:\경로\excel_compare
   INSTALL.bat
   RUN.bat
   ```

## 📝 사용 방법

1. **첫 번째 파일** 버튼 클릭 → 엑셀 파일 선택
2. **두 번째 파일** 버튼 클릭 → 엑셀 파일 선택  
3. **비교 시작** 버튼 클릭
4. 다른 셀은 **색상으로 표시**됩니다

## ❓ 문제가 생겼나요?

- **"Python을 찾을 수 없습니다"** → Python 설치 필요
- **"모듈을 찾을 수 없습니다"** → `INSTALL.bat` 실행
- **EXE가 실행 안 됨** → Windows Defender 예외 추가

자세한 내용은 `설치_및_실행_가이드.md`를 참고하세요.
